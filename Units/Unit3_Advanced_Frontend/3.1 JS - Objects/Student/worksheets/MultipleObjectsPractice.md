# 3.1 JS Objects – Worksheet
## Multiple Objects Practice

---

### Part A – Declare Objects

Write code (no need to run) to declare:

1. An object `book` with properties: `title`, `author`, `isbn`, `category`.

2. An object `course` with properties: `name`, `teacher`, `studentsCount`.

---

### Part B – Access and Update

Using your `book` object:

3. Write code to log the book title using dot notation.

4. Write code to change the `category` to a new value.

Using your `course` object:

5. Write code to log the teacher name using bracket notation.

6. Write code to add a new property `room` with any value.

---

### Part C – Check and Delete

7. Using `book`, write code to check if it has a property `"isbn"` using the `in` operator.

8. Write code to delete the `category` property using `delete`.

---

### Part D – List Keys and Values

9. Write code that logs all keys in `book` using `Object.keys(book)`.

10. Write a `for...in` loop that logs each key and value in `course`.
